URBANFLEET - STATIC HTML EXPORT
================================

This folder contains the static HTML version of the UrbanFleet website.

FILES:
------
- index.html      : Main HTML file
- favicon.png     : Website icon
- opengraph.jpg   : Social media preview image
- assets/         : All CSS, JavaScript, and images

HOW TO DEPLOY:
--------------
1. Upload all files and folders to your web hosting
2. Make sure index.html is in the root directory
3. That's it! The site will work on any static hosting

COMPATIBLE HOSTING:
-------------------
- Any web hosting with HTML support
- Hostinger, GoDaddy, Bluehost, etc.
- GitHub Pages, Netlify, Vercel (free options)
- cPanel file manager upload

NOTE:
-----
This is a static export. Form submissions will not save to a database.
Forms will show success messages but data won't be stored.

For full functionality with database, use the Node.js version.
